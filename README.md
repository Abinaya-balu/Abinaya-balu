# 🔥 Featured Projects

## 🧠 AI & Research Projects

### Severity Aware Depression Classification using DistilRoBERTa
Research-driven NLP system for detecting depression severity from social media text using transformer-based deep learning.

- Built contextual classification pipeline using DistilRoBERTa  
- Applied preprocessing, tokenization & fine-tuning  
- Improved prediction using sentiment-aware auxiliary features  
- Evaluated with accuracy, precision, recall & F1-score  

**Tech Stack:** Python, Hugging Face, PyTorch, NLP, Transformers  

---

### 🌍 CarbonLens — Carbon Analytics & Nudge Engine
A full-stack sustainability platform that tracks carbon emissions and generates personalized eco-friendly recommendations.

- Computes daily CO₂ emissions from commute & energy usage  
- Stores historical carbon trends  
- Generates personalized sustainability nudges  
- Built analytics dashboard for insights  

**Tech Stack:** MERN Stack, Analytics, Dashboard UI
